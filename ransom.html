Ransomware: not every virus is as scary as it is painted

List of ransomwares I'll write about:
- PUBG Ransomware
- Hitler Ransonware
- nRansom

Security researchers claim that encryption threats (Ransomware) are the most common causes of attacks on computers and mobile devices. The target of the attackers are not only ordinary users, but also companies and institutions, such as the city hall in Atlanta or a hospital in Indiana.

The first position on the list is taken by PUBG Ransomware. The name derives from the popular PlayerUnknown's Battlegrounds. The software encrypts the files on the victim's desktop, and adds the .PUBG extension to them. So what is the way to decrypt the files? Just play the game for one hour.
However, there are also other ways to restore files. You can simply rewrite the code from the welcome screen, or run the TslGame.exe file for 3 seconds, despite the required one hour. People who did not realize how this program works, probably it would have caused a lot of fear. In 2017, a similar threat, called RensenWare, was created, which in order to decrypt the files, ordered the victim to get 200 million points at the Lunatic level in Undefined Fantastic Object.

The next noteworthy malware is Hitler Ransomware. This software was discovered by AVG lab analyst - Jakub Kroustek. It is worth paying attention to the original spelling with an error,
Hitler-Ransonware. The program presents the figure of Adolf Hitler with the information, that in order to decrypt files victim should give the code for a 25 Euro Vodafone card.

Another unusual threat is the nRansom program, which has been detected by researchers from MalwareHunterTeam. Usually the Bitcoins are used to pay for the decryption of files due to identity protection, however, in the case of nRansomware, the tender card is to send your naked photos. Most probably, the program was created by a teenager. You can tell from the look of the application, the name of the music files and the e-mail address on which you need to send photos. The music file is called your-mom-gay.mp3, and in fact it's the song Frolic, written by Luciano Michelini. This is a composition also known from the Curb Your Enthusiasm show. 10 photos need to be sent to the email 1_kill_yourself_1@protonmail.com to get the code to decrypt files. The program does not fully work properly, because after entering the correct code and pressing the Unlock button, nothing happens. The way to close the program is to minimize it and end the process from the Task Manager.
